\# Level 3 - Smart Agent (Ritheeshwar K)



\## Overview



This project implements a simple intelligent agent that interacts with the LPI Sandbox.



The agent selects appropriate tools based on user query intent and combines multiple tool outputs to generate a structured response.



\---



\## Features



\- Intent-based tool selection

\- Multi-tool execution

\- Structured response generation

\- CLI-based interaction



\---



\## How It Works



1\. User inputs a query

2\. Agent analyzes the query

3\. Selects relevant tools

4\. Executes tools

5\. Combines outputs into a final response
