**PROJECT OVERVIEW **

I developed a simple LPI-based AI agent using Python.
The main idea of the project is to take a user’s question and provide useful suggestions by following the SMILE methodology.
The agent also uses a local LLM through Ollama to generate human-readable answers.
This project helped me understand how AI can process input step by step and give meaningful outputs instead of direct responses.
