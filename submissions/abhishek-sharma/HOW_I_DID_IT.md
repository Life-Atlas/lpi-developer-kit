# How I Did It - Abhishek Sharma

## Overview
I built a Multi Tool AI Agent using HTML, CSS, and JavaScript. The agent takes user input and performs different tasks based on keywords.

## Steps I Followed
1. Created the UI using HTML (input and output area)
2. Styled the interface using CSS
3. Used JavaScript to process user input
4. Implemented keyword-based logic to trigger tools

## Features
- Calculator: Performs basic calculations
- Time Tool: Displays current time
- Study Tips: Provides helpful study advice
- Motivation: Shows motivational messages

## Challenges
- Handling different types of user input
- Making sure correct tool runs

## Solutions
- Used if-else logic for keyword matching
- Kept the code simple and structured

## Conclusion
